bash

2. 验证推送是否成功
bash
# 检查状态
git status

# 查看提交历史
git log --oneline
3. 正常的 Git 工作流程
从现在开始，你可以使用标准的 Git 工作流程：

bash
# 1. 检查状态
git status

# 2. 添加更改的文件
git add .

# 3. 提交更改
git commit -m "描述你的更改"

# 4. 推送到 GitHub
git push
验证 GitHub 仓库：

如果你需要从头开始配置新项目：
初始化新仓库的完整流程：
bash
# 1. 进入项目目录
cd /path/to/your/project

# 2. 初始化 Git 仓库
git init

# 3. 添加文件
git add .

# 4. 提交更改
git commit -m "初始提交"

# 5. 添加远程仓库
git remote add origin git@github.com:Zz-zhuozhuo/仓库名.git

# 6. 推送到 GitHub
git push -u origin main
